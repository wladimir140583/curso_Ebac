# Atividade - Cadastro de Clientes com Web APIs

## Descrição
Formulário de cadastro de clientes com busca automática 
de endereço pelo CEP, utilizando Web APIs e armazenamento 
local dos dados cadastrados.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Fetch API (busca de endereço via ViaCEP)
- Web Storage API (localStorage)

## Como executar
1. Abra o arquivo `index.html` no navegador
2. Preencha os dados do cliente (Nome e Email)
3. Digite o CEP e clique na lupa para preencher o endereço automaticamente
4. Clique em **Cadastrar** para salvar os dados no localStorage

## Funcionalidades
- Busca automática de endereço pelo CEP (ViaCEP)
- Armazenamento de múltiplos cadastros no localStorage
- Campos limpos automaticamente após cadastrar
