# Aprendendo JS | POO - Criando um Parquímetro

## Descrição
Simulador de parquímetro desenvolvido com Programação Orientada a Objetos (POO) em JavaScript.
O sistema calcula o tempo de estacionamento e o troco com base no valor inserido pelo usuário.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (POO)

## Como executar
1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` no navegador
3. Digite o valor desejado e clique em calcular
